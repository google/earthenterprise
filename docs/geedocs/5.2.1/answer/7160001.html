Release notes: Open GEE 5.2.1

Open GEE 5.2.1 is actively under development

Supported Platforms

The Open GEE 5.2.1 release is supported on 64-bit versions of the following operating systems:

  • Red Hat Enterprise Linux version 6.x and 7.x, including the most recent security patches
  • CentOS 6.x and 7.x
  • Ubuntu 14.04 LTS and 16.04 LTS

Google Earth Enterprise 5.2.1 is compatible with Google Earth Enterprise Client (EC) version 7.1.5 and above.


If you're upgrading from GEE 5.1.3, you may have to reinstall the prerequisite libraries after uninstalling 5.1.3 but before installing 5.2.1.
