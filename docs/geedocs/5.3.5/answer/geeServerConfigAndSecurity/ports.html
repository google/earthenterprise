GEE Server uses the following ports:

  • Port 80 is used for publishing and streaming data to clients.

  • Port 443 is used instead of port 80 when GEE is configured for HTTPS.

GEE Fusion uses the following ports:

  • Ports 13031, 13032, and 13033 are used to communicate between the Fusion master and Fusion slaves.
