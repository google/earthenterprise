Uninstalling Google Earth EnterpriseÂ¶

Open GEE provides command line uninstall scripts. To uninstall Google Earth Enterprise Fusion and Server:

  1. Open a terminal window on your Linux workstation.

  2. Log in as root.

  3. Stop Google Earth Enterprise processes, if you have a previous version installed and it is running:

    /etc/init.d/gefusion stop
    /etc/init.d/geserver stop

  4. Navigate to the installation location (/opt/google/install).

  5. Run the scripts uninstall_server.sh and uninstall_fusion.sh.
