Configure font size in the GUIÂ¶

If you want to customize certain aspects of the Google Earth Enterprise Fusion GUI (such as menus, button labels, tables, lists, etc.), you can use a graphical configuration tool called Qt Configuration to customize the font size and other GUI features.

To customize the appearance of the Google Earth Enterprise Fusion GUI:

  1. On the command line, change directory to /opt/google/qt/bin/ and enter ./qtconfig.

    The Qt Configuration dialog appears.

  2. Click the tabs at the top of the right side of the window to view the available options, and use the help text on the left side for help with each tab.

    Note

    The Qt Configuration tool is not a Google product. For more information about Qt Configuration, select either of the options on the Help menu.
