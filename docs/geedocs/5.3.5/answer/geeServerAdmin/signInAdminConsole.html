Sign in to GEE Server Admin consoleÂ¶

To access the GEE Server Admin console for managing and publishing maps and globes:

  1. Go to myserver.mydomainname.com/admin, replacing myserver and mydomainname with your server and domain.

  2. Sign in with the default credentials:

    • Username: geapacheuser

    • Password: geeadmin

      Tip

      To reset the username and password: sudo /opt/google/gehttpd/bin/htpasswd -c /opt/google/gehttpd/conf.d/.htpasswd geapacheuser
