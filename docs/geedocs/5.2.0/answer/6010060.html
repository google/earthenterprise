Clearing the Google Earth EC cache

If you have signed in to a database on GEE Server with Google Earth EC previously, you may be unable to sign in again to the same database. An effective way of resolving this issue is to sign out of your server, clear your cache, and log back in.

To clear your cache:

  1. Select File > Server Sign Out.
  2. Select Google Earth EC > Preferences > Cache.
  3. Select Delete Cache File (only when logged out).
