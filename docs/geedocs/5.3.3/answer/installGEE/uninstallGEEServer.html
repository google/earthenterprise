Uninstalling GEE ServerÂ¶

GEE Server provides a command line uninstall script. To uninstall Google Earth Enterprise Server:

  1. Open a terminal window on your Linux workstation.

  2. Log in as root.

  3. Stop the Google Earth Enterprise Server, if you have a previous version installed and it is running:

    /etc/init.d/geserver stop

  4. Navigate to the installation location (/opt/google/install).

  5. Run the script uninstall_server.sh.
