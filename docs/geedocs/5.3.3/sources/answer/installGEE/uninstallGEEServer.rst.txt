
Uninstalling GEE Server

GEE Server provides a command line uninstall script. To uninstall
Google Earth Enterprise Server:

Open a terminal window on your Linux workstation.

Log in as root.

Stop the Google Earth Enterprise Server, if you have a previous
version installed and it is running:  /etc/init.d/geserver stop

Navigate to the installation location (/opt/google/install).

Run the script uninstall_server.sh.
