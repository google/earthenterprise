
Sample Data Files

This appendix provides a list of all of the sample data files included
when you install the tutorial data. The files are listed
hierarchically with folder names at the top of each section. Files
marked with an asterisk (*) are not used in the tutorial lessons. You
can use those files as you choose to explore the features of Google
Earth Enterprise Fusion.
