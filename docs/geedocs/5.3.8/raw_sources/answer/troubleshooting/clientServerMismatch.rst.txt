======================
Client/server mismatch
======================

.. container::

   .. container:: content

      .. rubric:: Error message

      ``Invalid protocol version (client/server mismatch)``
      
      .. rubric:: Description

      During a transition from version 3.x to a later version, you tried
      to use an older or newer version of the Fusion GUI or command line
      to connect to the system manager.

      .. rubric:: Resolution

      Ensure that you are using the same version of Fusion and GEE
      Server.
