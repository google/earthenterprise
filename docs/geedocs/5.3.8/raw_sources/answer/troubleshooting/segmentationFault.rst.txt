==================
Segmentation fault
==================

.. container::

   .. container:: content

      .. rubric:: Error message

      ``Segmentation fault``

      .. rubric:: Description

      The Segmentation fault error message typically indicates an issue
      with the operating environment, such as a software or hardware
      incompatibility.

      .. rubric:: Resolution

      Confirm that your hardware and software meet the requirements
      listed in the GEE documentation.
