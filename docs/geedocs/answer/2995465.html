Welcome to Google Earth Enterprise

Be sure to read the System requirements for Fusion and GEE Server before you begin.

  • If you need to install or access the documentation offline, see Download documentation for GEE 5.x.
  • Read Installation tips before to begin the installation process to install Fusion and GEE Server.
  • Read Portable User Guide for Mac OS, Windows, and Linux for instructions to install and use the Portable Server.

After installing GEE, you should complete the Fusion tutorial, which helps you learn Fusion quickly and allows you to create a sample project that you can use whenever you need to test the push/publishing process.
