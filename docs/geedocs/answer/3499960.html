Command line installer

Overview

You can install GEE Server with either the command line installer or the graphical user interface (GUI) installer. If you are installing on a server without X11, use the command-line installer. During installation, you are prompted for replies and guided through the process automatically.

If you want to install on multiple machines without user interaction, you can use the silent installer.

Before you begin

Google Earth Enterprise is available in two formats: as a download from Google.com servers, or on physical media in the form of DVDs. The instructions below are the same for both formats after the following setup has been completed:

If you downloaded the software package

Unpack the archived installer(s) to a temporary directory:

tar -zxvf GEEServer50.tar.gz -C /target/directory

If you ordered the DVD

Insert the distribution DVD, and mount the DVD drive.

Installing GEE Server

  1. Open a terminal window on your Linux workstation.
  2. Log in as root.
  3. Stop the Google Earth Enterprise Fusion system manager and Google Earth Enterprise Server, if you have a previous version installed and they are running:

    /etc/init.d/gefusion stop
    /etc/init.d/geserver stop


  4. Navigate to your DVD directory, or the directory to which you extracted the Google Earth Enterprise files, and run the Google Earth Server installer:

    ./InstallGEServer.sh

    If your DVD automounted without exec privileges, enter sh ./InstallGEServer.sh instead.

    The installation script starts running.

  5. Press Enter to continue. You can type Quit to exit the installation process or Back to return to the previous screen at any time.
  6. Press Enter to scroll through the Google License Agreement. Type Y to accept the terms.
  7. If the installation process detects a previous installation of Fusion, you have the option to create a backup of your previous configuration.

    Select Backup to create a back up of the existing configuration or Skip to continue the installation without creating a backup configuration file.

    The installer performs a backup of the existing configuration files to the following location:

    /var/opt/google/geserver-backups/date.time

    Note that the source volume(s) and asset root remain unchanged.

  8. Enter the applicable number for Typical, Minimal, or Custom installation, or press Enter to accept the default (Typical). If the installer determines it needs to upgrade any product you are installing, it prompts you to confirm the upgrade.
    • Typical installation includes Google Earth Enterprise Fusion and the Tutorial.
    • Minimal installation includes only Google Earth Enterprise Fusion.
    • Custom installation allows you to customize the components you install.
  9. The installer prompts you to create the user and group for Google Earth Enterprise Server. Press Enter to accept the default or enter your customized names.
  10. The installer prompts you to designate your publish root. Press Enter to accept the default (/gevol/published_dbs/), or enter the path you want to set for your publish root.
  11. The Pre-Installation Summary appears. Press Enter to continue.
  12. You can choose to start the Google Earth Enterprise Server services. To start them later, from the command line:

    /etc/init.d/gefusion start
    /etc/init.d/geserver start



Using the silent installer

You can use the silent installer to install GEE Server while unattended. This allows for automatic updates of the software as needed. You customize the installserver.properties template, located in the directory containing the installer files, for example, GEEServer-5.0/installer.

To use the silent installer:

  1. Locate the installserver.properties template file in the directory containing the installer files (for example, GEEServer-5.0/installer).
  2. Make a backup copy in another location, then edit the template to set the properties for your silent installation.
  3. Run the Google Earth Server silent installer: 
    ./InstallGEServer.sh installserver.properties

    You can customize the following properties for the silent installation. Set a value to 0 for the default, or 1 to customize the value:

    #Current Version of Google Earth Enterprise Server Already Installed

#--------------------------
PROMPT_USER_CHOSEN_OPTION=0

#Choose Products to Install
#---------------------------
CHOSEN_FEATURE_LIST=GEEServer,GEPlaces,SearchExample
CHOSEN_INSTALL_FEATURE_LIST=GEEServer,GEPlaces,SearchExample
CHOSEN_INSTALL_SET=Typical

#Create Google Earth Enterprise Server User and Group Names
#------------------------------
CUSTOM_USER_NAMES=0

#Specify Publish Root Location
#----------------------------
PUBLISHER_ROOT=/usr/local/google/gevol/published_db

#Install
#-------------------------------
-fileOverwrite_/opt/google/install/GoogleEarthServer_Uninstaller/Uninstall_Google_Earth_Enterprise_Server_5.0.lax=Yes
-fileOverwrite_/opt/google/share/fusion_splash.png=Yes
-fileOverwrite_/opt/google/gehttpd/conf/gehttpd.conf=Yes
-fileOverwrite_/opt/google/gehttpd/conf/magic=Yes
-fileOverwrite_/opt/google/gehttpd/conf/mime.types=Yes
-fileOverwrite_/opt/google/gehttpd/conf/ge_logging.conf=Yes
-fileOverwrite_/opt/google/gehttpd/htdocs/google_tm.png=Yes
-fileOverwrite_/opt/google/gehttpd/htdocs/index.shtml=Yes
-fileOverwrite_/opt/google/gehttpd/cgi-bin/apache_logs.py=Yes
-fileOverwrite_/opt/google/gehttpd/cgi-bin/globe_cutter_app.py=Yes
-fileOverwrite_/opt/google/gehttpd/cgi-bin/globe_info_app.py=Yes
-fileOverwrite_/opt/google/gehttpd/cgi-bin/glc_assembler_app.py=Yes
-fileOverwrite_/opt/google/gehttpd/cgi-bin/search/bounds_search.py=Yes
-fileOverwrite_/opt/google/gehttpd/cgi-bin/advanced_cutter.cfg=Yes
-fileOverwrite_/opt/google/share/searchexample/searchexample=Yes
-fileOverwrite_/opt/google/share/searchexample/searchexample_db.gz=Yes
-fileOverwrite_/opt/google/share/geplaces/geplaces_db.gz=Yes
-fileOverwrite_/opt/google/share/geplaces/geplaces=Yes
-fileOverwrite_/etc/init.d/geserver=Yes
-fileOverwrite_/etc/profile.d/ge-server.sh=Yes
-fileOverwrite_/etc/profile.d/ge-server.csh=Yes
-fileOverwrite_/etc/logrotate.d/gehttpd=Yes

#Start the Google Earth Enterprise Server Service
#--------------------------
PROMPT_FOR_START=1
