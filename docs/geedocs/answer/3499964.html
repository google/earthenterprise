Uninstalling

GEE provides a command line uninstall script. To uninstall Google Earth Enterprise Server:

  1. Open a terminal window on your Linux workstation.
  2. Log in as root.
  3. Stop the Google Earth Enterprise Server, if you have a previous version installed and it is running:

    /etc/init.d/geserver stop

  4. Navigate to the location at which you initially cloned the GEE git repo.
  5. Within the GEE git repo, navigate to earthenterprise/earth_enterprise/src/installer and run the script uninstall_server.sh.
