Release notes: GEE 5.1.2¶

Overview

GEE 5.1.2 is an incremental release of GEE 5.1. It contains several bug fixes in Fusion and GEE Server, and updates to third-party libraries.

Supported Platforms

The Google Earth Enterprise 5.1.2 release is supported on 64-bit versions of the following operating systems:

  • Red Hat Enterprise Linux versions 6.0 to 7.1, including the most recent security patches
  • CentOS 6.0 to 7.1
  • Ubuntu 10.04 and 12.04 LTS

Google Earth Enterprise 5.1.2 is compatible with Google Earth Enterprise Client (EC) version 7.1.5 and Google Earth Plug-in versions 7.0.1 - 7.1.5.1557 for Windows and Mac.
