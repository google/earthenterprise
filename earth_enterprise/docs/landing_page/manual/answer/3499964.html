Uninstalling

This section provides instructions for uninstalling all or some of the software provided in the Google Earth Enterprise software package. For this procedure, you use the same installer you used to install the software.

Note: As with installation, you can use either the command-line uninstaller or the graphical user interface (GUI) uninstaller to remove the Google Earth Enterprise Fusion and Server products.

To uninstall all or part of the Google Earth Enterprise software:

The uninstall script starts. Follow the onscreen instructions to remove the Google Earth Enterprise Fusion or Server products.

  1. Open a terminal window on your Linux workstation.
  2. Log in as root.
  3. Stop the Google Earth Enterprise Server, if you have a previous version installed and it is running:

    /etc/init.d/geserver stop

  4. Navigate to the /opt/install/GoogleEarthServer_Uninstaller/ directory.

    To uninstall using the Google Earth Enterprise Server DVD, insert the DVD and mount the drive. The uninstallers listed below are available at the top level of the DVD media.

  5. To uninstall Google Earth Enterprise Server: ./Uninstall_Google_Earth_Enterprise_Server_GUI.sh // GUI uninstaller ./Uninstall_Google_Earth_Enterprise_Server.sh // Command line uninstaller
