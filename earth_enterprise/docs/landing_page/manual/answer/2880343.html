GEE downloads to Mac

Macs that run OS X Mountain Lion do not yet list the Google Support Portal as an "identified developer," so they place Portal downloads in the trash as a quarantine measure. If you download EC or Earth plug-in from the Support Portal to a Mac running Mountain Lion with security set to "maximum", you'll need to move them into the appropriate directory after download. For more information, see Apple support.
